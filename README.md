# 幸運號碼抽選機 (Lucky Draw)

這是一個高質感的網頁版數字抽選機，完全使用 HTML、CSS 與 JavaScript 開發，無需安裝任何軟體即可執行。

## 🌟 功能特色
- **1-99 號碼亂數抽選**：公平隨機產生號碼。
- **防止重複**：在這個回合內，已經抽出的號碼不會再次出現。
- **累積顯示**：抽出的號碼會保留在畫面上，直到您手動清除。
- **高質感介面**：深色玻璃擬態 (Glassmorphism) 設計，搭配流暢動畫。

## 🚀 如何啟動
1. 進入 `DrawNum` 資料夾。
2. 雙擊 **`index.html`** 檔案，即可在您的預設瀏覽器中開啟。

## 📖 操作說明

### 1. 設定數量
在畫面中央的輸入框中，輸入您想要一次抽選的號碼數量。
- **範圍**：一次最少 1 個，最多 5 個。

### 2. 開始抽選 (DRAW)
點擊 **`DRAW`** 按鈕。
- 號碼球會以動畫方式跳出。
- 若您再次點擊 `DRAW`，新的號碼會累積在後方，且保證不與前面的號碼重複。
- 當 1-99 號全部抽完時，系統會提示您無法再抽。

### 3. 清除重置 (CLEAR)
點擊 **`CLEAR`** 按鈕。
- 畫面上的所有號碼球將被清除。
- 抽選紀錄歸零，下一球可能再次抽到任何號碼。

## 🛠️ 技術規格
- **HTML5** (語意化結構)
- **CSS3** (Flexbox, Animations, Glassmorphism)
- **JavaScript (ES6+)** (邏輯控制, Set 資料結構)
